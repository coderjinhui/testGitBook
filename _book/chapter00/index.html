前置

前置

通常来说，学习 Angular 之前需要会很多前置技能，就像装备合成路线一样，高级装备总是需要先合成一些低级材料，有了这些基础才能合成更加强大的装备。

需要的技能

*表示必须掌握

  • *HTML
  • *CSS
  • *JavaScript
  • Typescript

在 Angular 中，这些前置技能并没有那么严格的需求，总的来说需要的技能和其他框架类似，其中必须的还是前端最基础的那些东西 HTML, CSS, JavaScript

如果你了解并使用过 typescript 那就更完美了。如果不会也没关系，它不是必需品，缺少了 typescript 并不代表 Angular 就完全学不会了，我们可以在.ts文件中直接写 JavaScript，即便是这样写会少了很多 typescript 语言特性，但它依然能够运行。

更为重要的

学习过程中，主观能动性大于一切，因此更为重要的是需要放平心态，不要轻言放弃。我有几个朋友就因为 typescript 不会而放弃学习 Angular，说实话会不会 typescript 并不会特别影响 Angular 的学习，不会 typescript 顶多会让你无法使用那些相当不错的语言特性，往极端一点说也只是 Angular 代码写的比较烂，仅此而已。

我深信一点: 你永远叫不醒一个装睡的人。学习也同样如此，你永远学不会你不想学的东西。
